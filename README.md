# MiraiApi - Miku Random Image Api

## 介绍
MiraiApi，意即「**Mi**ku **Ra**ndom **I**mage Api」，是一个随机图 Api 程序。

主要目的是将**本地**中的图片以*单一链接*的形式随机展示出来。

## 部署
先将程序克隆到本地中
```
git clone https://github.com/mouyase/MiraiApi.git
```

打开程序文件夹

```
cd MiraiApi
```

适当修改`docker-compose.yml`文件中的配置，随后启动
```
docker compose up -d
```
